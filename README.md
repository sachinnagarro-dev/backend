# Conversation Session Service

Backend Take-Home Assignment for Voice AI Platform.

## Setup Instructions

1. Ensure Node.js (v18+) and npm are installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start MongoDB:
   ```bash
   docker run -d --name mongodb -p 27017:27017 mongo:latest
   ```
4. Run the application:
   ```bash
   npm run start:dev
   ```

The server will start on port 3000.

## How to run the project

- Development: `npm run start:dev`
- Production: `npm run start:prod`

## Assumptions Made

- MongoDB is running locally on port 27017.
- No authentication required as per constraints.
- Simple pagination with limit/offset.
- Payload in events is a flexible JSON object.

## API Endpoints

- `POST /sessions` - Create or upsert session
- `POST /sessions/:sessionId/events` - Add event to session
- `GET /sessions/:sessionId` - Get session with paginated events
- `POST /sessions/:sessionId/complete` - Complete session
